# My Website
This site is developed using [Hugo](https://gohugo.io/) with the [Hextra](https://imfing.github.io/hextra/) theme. Make sure you have the required [prerequisites](https://imfing.github.io/hextra/versions/latest/docs/getting-started/), then run
`hugo server --buildDrafts --disableFastRender` to preview the site locally.
